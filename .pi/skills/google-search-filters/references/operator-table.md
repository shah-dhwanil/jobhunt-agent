# Google Search Operator Quick Reference

A flat, copy-paste-ready list of all working Google search operators.

---

## All Working Operators (as of 2026)

| Operator | Syntax | Example |
|---|---|---|
| Exact phrase | `"phrase"` | `"best coffee maker"` |
| OR | `term1 OR term2` | `iphone OR android` |
| OR (pipe) | `term1 \| term2` | `iphone \| android` |
| AND | `term1 AND term2` | `python AND tutorial` |
| Exclude | `-term` | `jaguar -car` |
| Wildcard | `*` | `"best * for beginners"` |
| Grouping | `(term1 OR term2) term3` | `(ipad OR iphone) apple` |
| Site filter | `site:domain` | `site:github.com` |
| Exclude site | `-site:domain` | `-site:pinterest.com` |
| Related sites | `related:domain` | `related:techcrunch.com` |
| In title | `intitle:word` | `intitle:guide` |
| All in title | `allintitle:words` | `allintitle:python beginner tutorial` |
| In URL | `inurl:word` | `inurl:resources` |
| All in URL | `allinurl:words` | `allinurl:case study` |
| In body text | `intext:word` | `intext:"open source"` |
| All in body text | `allintext:words` | `allintext:climate data analysis` |
| File type | `filetype:ext` | `filetype:pdf` |
| File type (alt) | `ext:ext` | `ext:xlsx` |
| Before date | `before:YYYY-MM-DD` | `before:2024-01-01` |
| After date | `after:YYYY-MM-DD` | `after:2023-06-01` |
| Definition | `define:word` | `define:blockchain` |
| Cache | `cache:url` | `cache:wikipedia.org` |
| News source | `source:outlet` | `source:nytimes` |
| Unit convert | `X in Y` | `100 USD in EUR` |
| Weather | `weather:location` | `weather:tokyo` |
| Stocks | `stocks:ticker` | `stocks:msft` |
| Movie info | `movie:title` | `movie:dune` |
| Map | `map:location` | `map:brooklyn` |

---

## Unreliable (use cautiously)

| Operator | Syntax | Notes |
|---|---|---|
| Number range | `X..Y` | `headphones $50..$200` — inconsistent |
| Proximity | `AROUND(X)` | `AI AROUND(3) regulation` — hit or miss |
| Anchor text | `inanchor:` | Inconsistent across queries |
| Location | `loc:` | Limited functionality |
